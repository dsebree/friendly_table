Goal of this plugin:

Make a reusable jQuery plugin to detect browser size then reorganize table data into a mobile friendly view.

Requirements:

  • jQuery
  • Bootstrap

Friendly Table Plugin:

  • jquery.ft.plugin.js
  • ft.styles.css

Wireframe:

This image illustratess my initial idea going into building the plugin. On mobile devices the plugin takes the text from each TH (heading) and then appends it to the corresponding TDs. Each row of TDs is organized vertically and the user can expand/collapse each row to see the data. I believe this is more consistent with modern UX design, rather than zooming and scrolling to see table data in more detail.

Zoom in

Friendly Table plugin in action:

$(".friendly").friendlytable();

(Mobile version activates at window width of 768px)
